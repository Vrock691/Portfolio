EDT IUT Périgueux

EDT IUT Périgueux est une application flutter disponible sur le play store. Elle est dédiée aux étudiant de l'iut de Périgueux. Son but est de récupèrer les données d'emploi du temps depuis le site officiel et les organiser dans un widget de calendrier facilement utilisable par l'utilisateur.

Voir sur le Play Store

Pourcentage

Pourcentage est une application flutter à code source ouvert disponible sur Github et prochainement sur le Play Store. Son objectif est de calculer le prix d'un produit après une réduction en pourcentage.

Voir sur Github

Spotify sync

Spotify sync était un script javascript qui synchronisait les playlist spotify et deezer de son utilisateur. En dialogant avec les API des deux services, il comparait les tracklist et ajoutait de part et d'autres les titres manquants.

Code non disponible

Pluméo

Pluméo était un projet de messagerie sécurisé mené pendant 2 ans avec Shawiizz. Entièrement codé pour le web, elle était multiplateforme et prennait en charge les messages textes, audios et les appels audios et vidéos par WebRTC. Ce projet est toujours en cours de développement mais uniquement dans le cadre des études de Shawiizz.
